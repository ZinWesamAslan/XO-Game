# XO Game (Tic-Tac-Toe) - Windows Forms Application

## Project Description
A simple desktop application built with C# and Windows Forms that provides the classic XO (Tic-Tac-Toe) game for two players (Player vs Player).  
The application allows you to choose the number of matches (1, 3, or 5) and determines the winner of each match. At the end, it declares the overall champion and displays their name along with a celebratory image.

---

## Technologies Used
- Programming Language: C#
- Framework: .NET Framework (or .NET Core/5+ with Windows Forms)
- Development Environment: Visual Studio 2022 (or any version supporting Windows Forms)

---

## Key Features
- Select the number of matches before starting: 1, 3, or 5.
- Interactive graphical interface with a 3x3 grid.
- Two-player gameplay on the same device.
- Determines the winner of each match immediately after it ends.
- Tracks cumulative scores after each match.
- Declares the overall champion at the end of all matches, displaying a custom image for the winner.
- Option to restart the game or start a new session.

---

## Game Rules (XO / Tic-Tac-Toe)
- The game is played on a 3x3 grid.
- The goal is to get three marks in a row (horizontally, vertically, or diagonally).
- If the grid fills up without a winner, the match is considered a draw and no points are awarded.
- After each match, the winner receives one point in the score tracker.
- After all matches are played, the player with the higher score is declared the overall champion.
